## Run 

Clone the project

```bash
  git clone https://github.com/harrietfiagbor/Info-retrieval-agent-chat.git
```

Go to the project directory

```bash
  cd Info-retrieval-agent-chat
```

Install dependencies

```bash
  pip install -r requirements.txt
  cd src/
```

Start the chat bot

```bash
  python app.py
```
